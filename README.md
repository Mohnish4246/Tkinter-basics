# 🖥️ Tkinter GUI Application

A simple Python GUI application built using **Tkinter**. This project demonstrates how to create interactive desktop applications with buttons, inputs, and basic functionality.

---

## 📌 Features

- User-friendly graphical interface  
- Input fields and buttons  
- Event handling (button clicks, user actions)  
- Lightweight and easy to run  
- Beginner-friendly project  

---

## 🛠️ Technologies Used

- Python 🐍  
- Tkinter (built-in GUI library)
